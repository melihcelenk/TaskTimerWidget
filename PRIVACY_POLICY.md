# Privacy Policy for TaskTimerWidget

**Last Updated: October 30, 2025**

## Overview

TaskTimerWidget ("the App") is a simple task timer application for Windows. This Privacy Policy describes how the App handles your information.

## Information Collection and Use

**TaskTimerWidget does NOT collect, transmit, or share any personal data.**

### Local Data Storage

The App stores the following information **locally on your device only**:

- **Task Names**: The names you give to your tasks
- **Time Tracking Data**: Elapsed time for each task
- **Task Settings**: Order and state of your tasks

All data is stored in a local JSON file at:
```
%LOCALAPPDATA%\TaskTimerWidget\Data\tasks.json
```

This data **never leaves your device** and is **not transmitted to any servers**.

### No Internet Connection Required

TaskTimerWidget operates entirely offline and does not:
- Connect to the internet
- Collect analytics or telemetry
- Track user behavior
- Share data with third parties
- Use cookies or similar tracking technologies

### Data Control

You have complete control over your data:
- All data is stored locally on your device
- You can delete all data by uninstalling the application
- You can manually access and modify the data file if needed

## Children's Privacy

TaskTimerWidget does not collect any personal information from anyone, including children under the age of 13.

## Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated "Last Updated" date.

## Contact

If you have any questions about this Privacy Policy, please create an issue on our GitHub repository:
https://github.com/melihcelenk/TaskTimerWidget

## Consent

By using TaskTimerWidget, you consent to this Privacy Policy.

### Yerel Veri Depolama

Uygulama aşağıdaki bilgileri **yalnızca cihazınızda yerel olarak** saklar:

- **Görev Adları**: Görevlerinize verdiğiniz isimler

```
%LOCALAPPDATA%\TaskTimerWidget\Data\tasks.json
```
